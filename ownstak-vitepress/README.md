# OwnStak VitePress Example

This is an example VitePress documentation site designed to be deployed on OwnStak.

## Quick Start

### Setup and Install

```bash
cd ownstak-vitepress
npm install
```

### Development

```bash
npx ownstak dev
```

Open your browser to `http://localhost:3000`

### Deploy to OwnStak

```bash
npx ownstak deploy
```
