# OwnStak Astro Example

This is an example Astro application designed to be deployed on OwnStak.

## Quick Start

### Setup and Install

```bash
cd ownstak-astro
npm install
```

### Development

```bash
npx ownstak dev
```

Open your browser to `http://localhost:3000`

### Deploy to OwnStak

```bash
npx ownstak deploy
```
