# OwnStak Next.js Example

This is an example Next.js application designed to be deployed on OwnStak.

## Quick Start

### Clone and Install

```bash
cd ownstak-nextjs
npm install
```

### Development

```bash
npx ownstak dev
```

Open your browser to `http://localhost:3000`

### Deploy to OwnStak

```bash
npx ownstak deploy
```
